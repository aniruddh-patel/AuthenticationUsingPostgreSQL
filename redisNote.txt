| Type                         | Description                                                                         | Example Commands                   |
| ---------------------------- | ----------------------------------------------------------------------------------- | ---------------------------------- |
| **String**                   | Binary-safe strings (text, numbers, or serialized data)                             | `SET`, `GET`, `INCR`, `APPEND`     |
| **Hash**                     | A map/dictionary of fields and values                                               | `HSET`, `HGET`, `HGETALL`          |
| **List**                     | Ordered list of strings (linked list)                                               | `LPUSH`, `RPUSH`, `LPOP`, `LRANGE` |
| **Set**                      | Unordered collection of unique strings                                              | `SADD`, `SMEMBERS`, `SREM`         |
| **Sorted Set (ZSet)**        | Set of unique strings ordered by a score                                            | `ZADD`, `ZRANGE`, `ZREM`           |
| **Vector Set**               | (Redis 7.2+ / Redis Stack) Stores numeric vectors for similarity search (AI/ML use) | `FT.CREATE`, `VECTOR` fields       |
| **Stream**                   | Append-only log data structure                                                      | `XADD`, `XRANGE`, `XREAD`          |
| **Bitmap**                   | Bit-level operations on strings                                                     | `SETBIT`, `GETBIT`, `BITCOUNT`     |
| **Bitfield**                 | Treats strings as arrays of integers at the bit level                               | `BITFIELD`                         |
| **Geospatial**               | Stores and queries coordinates                                                      | `GEOADD`, `GEORADIUS`, `GEODIST`   |
| **JSON**                     | (RedisJSON module) Stores and manipulates JSON objects                              | `JSON.SET`, `JSON.GET`             |
| **Probabilistic Data Types** | (RedisBloom module) For approximate set membership/counting                         | `BF.ADD`, `BF.EXISTS`, `CF.ADD`    |
| **Time Series**              | (RedisTimeSeries module) Stores and queries timestamped values                      | `TS.ADD`, `TS.RANGE`               |
